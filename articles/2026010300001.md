# なぜ作成したのか

- 自宅の蔵書部屋がもう人の住めない腐海となってしまったのでなんとかしたい
- まずは陳腐化した技術書の売却を考えるも、あんまり値段もつかない
  - 実際紙媒体として購入したときには内容がすでに古くなっているケースも多々ある
- そろそろ技術書は電子化すべき時期かもしれない
- オライリー社のサブスクも知っていたが、ちょうど年末に「TechLib」というサービスがリリースされたので試してみる

# やりたいこと
- 毎月数万の費用が発生している技術書・資格対策書籍が代替できるかを検証する

---

# TechLib とは

https://techlib.co.jp/

> - ITエンジニア向けの技術書籍読み放題サブスクリプションサービス
> - 2025年12月に正式リリースされ、インプレスやエムディーエヌコーポレーションなどの主要出版社が参加
> - 月額3,080円で、個人は10日間、法人は15日間の無料トライアルが利用可能
> - プログラミング言語、AI、クラウドサービス、セキュリティなど幅広いジャンルの技術書が提供されており、全文検索機能やソースコードのコピー＆ペースト機能を具備

# 所感

- 蔵書数が少ない（いまのところ欲しい本がほとんどない）
  - 自宅蔵書からの移行を考えると、オライリー、技術評論社等の書籍がないのは結構致命的
  - 出版社として登録されていても、すべての書籍が対象というわけではない（マイナビ出版を例に自宅の蔵書からサンプリングすると、「生成AIによるサイバーセキュリティ実践ガイド」が登録されてない など）
- 全文検索、コピペの機能は書籍派からするととても便利
  - AtlasやClaude in Chromeなどのエージェントを併用できるとさらに用途が増しそう
- 著作権的に非営利での活用はOKだけど、自分（自社）のものとして権利を主張するのはもちろんダメ
  - とはいえコーディングや執筆、制作を生成AI、エージェント併用すると **意図せず抵触する** 可能性は高い

# 有償契約する？
- 正直現時点では、UXとしては好ましいとしても蔵書量の点で自宅蔵書の代替にはならない
- しかし著作権が絡むことから、出版社、著者との調整が極めて困難なサービスであることは容易に想像できる
- そして課金が期待されなければサービスは拡大・成長しない
- 今後のサービス展開、蔵書の追加への期待を込めて半年程度課金してみようと思う
